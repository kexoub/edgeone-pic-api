Edgeone 随机图片 API

简洁、快速、免费。通过简单的 GET 请求获取横/竖屏随机图片，自动识别设备返回合适图片。

源码与支持

项目托管： Elvish064/edgeone-pic-api

使用方法

通过简单的 HTTP GET 请求即可获取随机图片，支持以下参数：

GET
/pc

获取横屏随机图片，适合桌面设备使用。

GET
/pe

获取竖屏随机图片，适合移动设备使用。

GET
/ua

根据用户设备类型自动选择图片类型（桌面设备返回横屏，移动设备返回竖屏）。

使用示例

获取横屏图片

/pc

获取竖屏图片

/pe

根据设备自动选择

/ua

根据设备自动选择4个

/api/?type=ua&format=text&count=
